# CSE360-Student-QA-System
## Author
James Eshun

## Dependencies
Using Eclipse IDE, an Eclipse workspace must be set up with JavaFX23 and JDK23 libraries on the Modulepath, and H2 library on the Classpath, for the Build Path configuration.

## How to download
To view  application download this repository into a zip file, open it into a setup Eclipse workspace as suggested above, then run **src/application/TestsHW3.java**.
